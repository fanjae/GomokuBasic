# 콘솔 환경에서 구현한 2인용 오목
- C++로 구현한 Windows Console 기반 2인용 오목 게임입니다.

## 주요 기능
- 15x15 오목판 출력
- 키보드 방향키를 이용한 커서 이동
- Enter 키를 이용한 돌 배치
- 2인용 턴제 진행
- 가로, 세로, 대각선 방향 승리 판정
- 돌이 놓인 위치에 대한 입력 방지
- 게임 방법 및 크레딧 메뉴 제공

## 조작 방법
- ↑ ↓ ← → : 커서 이동
- Enter : 돌 배치

## 실행 환경
- C++
- Windows Console
- Visual Studio 2022

## 실행 방법
- Visual Studio 2022에서 프로젝트를 열고 빌드 후 실행합니다.
