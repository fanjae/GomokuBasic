# 콘솔 환경에서 구현한 2인용 오목

## 주요 기능
- 15x15 오목판 출력
- 키보드 방향키를 이용한 커서 이동
- Enter 키를 이용한 돌 배치
- 2인용 턴제 진행
- 가로, 세로, 대각선 방향 승리 판정
- 돌이 놓인 위치에 대한 입력 방지
- 게임 방법 및 크레딧 메뉴 제공

## 조작 방법
↑ ↓ ← → : 커서 이동
Enter : 돌 배치

## 실행 환경
- C++
- Windows Console
- Visual Studio 2022

## 개발 목적
- C++ 기본 문법, 배열, 반복문, 조건문, 함수 분리, 콘솔 입력 처리, 간단한 게임 로직 구현을 연습하기 위해 제작했습니다.

## 코드 구성
- menu() : 메인 메뉴 입력 처리
- PlayGame() : 게임 루프 및 턴 진행
- PrintBoard() : 현재 보드 상태 출력
- WinCheck() : 승리 조건 검사
- IndexSafeCheck() : 보드 범위 검사
